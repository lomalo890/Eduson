Создание словаря и добавление новых элементов
c = {
    'a': 1,
    'b': 2,
    'c': 3
}
c = dict
(
    [
        (1, 'a'), 
        (2, 'b'), 
        (3, 'c')
    ]
)

Поменять значение ключа
c[1] = 'abs'

Можно ещё так сделать ключ
a[(q, 2, 3)] = 'my tuple'
Будет
(q, 2, 3): 'my tuple'

Есть метод hash
Список не хешируется
В словаре не может быть списка, поскольку список не имеет хеша + в словаре могут быть только неизменяемые данные.



Очистка словаря
c.clear()

Получение элементов
d.get(1234) -- если такого ключа нет, то get не выбрасывает исключение, как d[1234]
Второй аргумент get получает значение при отсутствии ключа

Получение пар элементов
d,items()

Получение ключей и значений
d.keys(), d.values()

Соединять словари можно через d.update(d2), но значение старого ключа не препишется новым
{**d, **d2}
d | d2

Удаление элементов
d.pop(100) - key
d.popitem() - remove last
del d1[1]
